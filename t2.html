Membuat program "hello world" dalam berbagai bahasa

1. Python :

2. Java

3. C++

#include int main() { std::cout << "Hello, World!" << std::endl; return 0; }

4. JavaScript

console.log("Hello, World!");

5. Ruby

puts "Hello, World!"

6. Go

package main import "fmt" func main() { fmt.Println("Hello, World!") }

7. PHP
